# 指令集和解释器

Java 指令的操作码占1字节，总共可表示 2^8 种指令。

Java 指令是可变长指令。

指令：操作码、助记符

指令类型：
* 常量指令
* 加载指令
* 储存指令
* 操作数栈指令
* 数学指令
* 转换指令
* 比较指令
* 控制指令
* 引用指令
* 拓展指令
* 保留指令
  * 虚拟机内部使用，不能出现在`.class`文件中
    * 202，0xCA，breakpoint，断点
    * 254，0xFE，`impdep1`
    * 255，0xFF，`impdep2`

Java 虚拟机解释器大致工作流程：

do
  计算pc，获取pc中的opcode
  如果有operands则读取operands
  执行指令
while (还有下一条指令)

拓展指令一共6条，jsr 和 jsr_w 在高版本中已经被禁用，不做考虑。
